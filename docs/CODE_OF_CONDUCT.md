# Code of Conduct

Participation requires respectful, specific, evidence-based collaboration. Harassment, threats, discrimination, doxxing, credential exposure, and personal attacks are unacceptable in project spaces.

Maintainers may edit or remove harmful content and temporarily or permanently restrict participation when necessary to protect contributors and users. Report conduct concerns privately through the maintainer contact on the GitHub profile rather than a public issue. Reports will be handled as confidentially as practical, with conflicts of interest disclosed and avoided.
